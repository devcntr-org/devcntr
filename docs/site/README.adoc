= Documentation Site Build

This directory contains the Antora configuration and build tools for generating the Dev Center documentation site.

== Building the Documentation

Navigate to the docs/site folder

[source,bash]
----
cd docs/site
----

Install the Antora package and build

[source,bash]
----
pnpm install
pnpm run docs:build
----

The generated site will be output to `docs/build/site/`.

== Structure

* `antora-playbook.yml` - Antora playbook configuration (defines how to build the site)
* `package.json` - Node.js dependencies and build scripts (includes simple-icons)

The playbook points to `docs/modules` for the source content and outputs to `docs/build/site/`.
